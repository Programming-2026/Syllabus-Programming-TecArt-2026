## ğŸ§© Jenis Algoritma dalam Competitive Programming

### Prefix Sum
**Deskripsi:** Teknik pra-pemrosesan untuk query range cepat.  
**Capaian:** Menyelesaikan soal range query secara efisien.

---

### Sieve of Eratosthenes
**Deskripsi:** Algoritma untuk mencari bilangan prima hingga n.  
**Capaian:** Menggunakan sieve untuk optimasi soal number theory.

---

### Ternary Search
**Deskripsi:** Teknik pencarian untuk fungsi unimodal.  
**Capaian:** Menentukan kapan ternary search lebih tepat dari binary search.

---

### Dynamic Programming
**Deskripsi:** Teknik memecah masalah menjadi submasalah yang tumpang tindih.  
**Capaian:**
- Mengidentifikasi overlapping subproblem
- Menggunakan memoization & tabulation

---

### Euclidean Algorithm
**Deskripsi:** Algoritma untuk mencari GCD dua bilangan.  
**Capaian:** Menggunakan GCD dalam problem matematika CP.

---

### Greedy Algorithm
**Deskripsi:** Memilih solusi lokal terbaik di setiap langkah.  
**Capaian:** Menentukan kelayakan greedy pada suatu masalah.

---

### Binary Search Tree (BST)
**Deskripsi:** Pohon biner terurut untuk pencarian dinamis.  
**Capaian:** Mengimplementasikan operasi dasar BST.

---

### Binary Search
**Deskripsi:** Pencarian efisien pada data terurut.  
**Capaian:** Menggunakan binary search dan binary search on answer.

---

### Quick Sort
**Deskripsi:** Algoritma sorting berbasis divide and conquer.  
**Capaian:** Memahami pivot, partisi, dan kompleksitasnya.

---

### Dijkstraâ€™s Algorithm
**Deskripsi:** Algoritma jalur terpendek pada graph berbobot positif.  
**Capaian:** Menentukan shortest path dari satu sumber.

---

### Breadth First Search (BFS)
**Deskripsi:** Traversal graph secara level.  
**Capaian:** Menggunakan BFS untuk shortest path pada unweighted graph.

---

### Depth First Search (DFS)
**Deskripsi:** Traversal graph secara mendalam.  
**Capaian:** Menggunakan DFS untuk eksplorasi graph dan tree.
